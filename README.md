# Smart Attendance System

A smart attendance management system built using AI and web technologies.  
This project is designed to improve classroom attendance by using facial authentication, anti-spoofing, emotion detection, and GPS-based validation.

## Features

- Face recognition based student identification
- Anti-spoofing for fake face prevention
- Emotion detection during attendance capture
- GPS-based location validation
- Student dashboard
- Admin dashboard
- AI assistant support
- Attendance analytics and records

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite
- OpenCV

## Modules

### Admin Module
- Manage students
- Create and manage class sessions
- Monitor attendance records
- View analytics
- Control attendance settings

### Student Module
- Self attendance
- View attendance history
- Weekly class schedule
- Personal analytics
- Correction requests

## Project Objective

The main objective of this project is to create a secure, intelligent, and user-friendly attendance system that reduces manual work and improves accuracy in attendance tracking.
